# Текстуры и декор

На сервере можно менять внешний вид предметов через специальное меню.

![Меню текстур](/wiki/smodelmenu.webp)

## Как использовать

Возьмите предмет в руку и пропишите команду `/smodel`. Откроется меню с доступными текстурами.

## Поддерживаемые предметы

- **Все инструменты** начиная с железных
- **Шлемы** начиная с железного — меняются на декоративные шапки
- **Зелья**
- **Палка**
- **Молот**
- **Тотемы**

## Мебель

Для декораций используется **Железный слиток**. Через меню выберите модель мебели, затем вставьте полученный слиток в **рамку** чтобы установить её.
